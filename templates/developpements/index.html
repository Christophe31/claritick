Consultation

Les entêtes de colonnes abbrégés sont documentés par un tooltip.

Liste des développements

Depuis la consultation des développements, un clic-droit sur un développement permet de le modifier directement, le même raccourci est disponible pour les groupes de développements.

La description complète d'un développement est accessible en tooltip. Idem pour un groupe de développement

Une date de sortie prévisible est calculée pour chaque développement en fonction des développements "plus importants", situés au dessus de celui-ci.

Une icône d'alerte apparaît si la date de sortie prévisible dépasse une date d'engagement ou une date de sortie de version pour laquelle ce développement est requis.

Liste des versions

Dans la consultation des versions, pour chaque version est calculée une date de sortie "retardée" permettant d'inclure toutes les fonctionnalités requises dans cette version.

Si une version est déjà sortie, un clic sur l'icône correspondant permet d'avoir la liste des fonctionnalités incluse dans cette version.

Dans le cas contraire, un clic sur le numéro de version affiche les développements qui ont été explicitements notés comme étant requis pour cette version. Un clic sur la Date de Sortie Prévue affiche la liste des fonctionnalités qui seront réellement incluses dans la version si on la sort à cette date là. Et enfin, un clic sur la Date de Sortie Calculée (ou Complète) affiche la liste des fonctionnalités supplémentaires incluses si la date de sortie est retardée.
